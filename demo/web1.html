Hello World

作为程序员来说，我们对于Hello World这个短语十分熟悉，这是因为在新手期的学习中，又或者是平时开发调试中，都习惯性地来敲打这句话来检验自己的代码

中文名 你好，世界 类型 程序代码
外文名 Hello World 出处 《The C Programming Language》
创造者 Brian Kernighan 场景 计算机编程

下面就来演示各种语言版本的Hello World的输出实现

  • Java
  • Python
  • C++
  • Javascript
  • 更多语言

Java

public class Hello{
    public static void main(String[] args){
        System.out.println("Hello World");
    }
}

Python

print('Hello World')

C++

#include<iostream>
using namespace std;
int main(void){
    cout<<"Hello World";
    return 0;
}

JavaScript

console.log('Hello World');
